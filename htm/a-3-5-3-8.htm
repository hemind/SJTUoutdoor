标  题: 说说自己的想法
发信站: 饮水思源 (2007年05月22日18:29:43 星期二)
姐姐说得是啊，其实当初进入协会就是想在这里找到自己的乐趣，曾经看到过困难，曾经
想过要退缩，但也曾经被前辈们所感动。
仍然记得第一次代表协会参加比赛时的激动，真的很感激协会，也很感激当初信任我们的
学长。所以没有半点对他们不尊重的意思。
并没有任何想要排斥某人的意思，我们也不认为自己所谓的反击是有天大的道理，只是每
个人都有自己的想法，这也本无可厚非的。
本来户外就是一个让人找到自己快乐的地方，大家偶尔吵两句，但相信凭着户外人的胸怀
，相信这也不算什么。
既然我们已经接班了，我们会做好自己的工作，当然，每一代人都有自己的理念，也希望
学长们能够理解。或许在多年以后我们也会碰到今天学长们的问题，但我想，只要抱一颗
喜爱户外的心，包容，也没有什么大不了的。

再次申明，这里仍旧是各位喜爱户外的朋友们的家园，outdoor依旧欢迎各位，野协永远是
大家的野协。

: 为还有很多已经毕业的版友，或者本来就不属于协会这个圈子的版友，他们都属于这个

: 大平台。
: 那么协会的同仁们如何对待版上的不同的声音呢？
: 既然这是个大平台，就要允许大家各抒己见，不论对错。
: 既然平台给了所有人这个自由
: ，就请拥有这个权力的所有人尊重版面上的其他所有人。
: 批评的，抑或被批评的，都请不要看到反驳的帖子就脸红脖子粗。请冷静的想一下，别

: 说实话，我以前也是一旦被人批评，一旦被人指责，就立即给自己找理由，找借口。每
