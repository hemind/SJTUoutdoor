选择登山包的几点中肯建议（搜集整理） 
一个选择正确的背包，会成为你旅途中最得力的助手。因为她知道如何将你需要背负的重
量降到最低限度，从而节省你的体力。经过下面的几个步骤，你会知道如何去选择一个适
合你的背包。 

Step1 明确你旅行的类型 

这是个非常关键的步骤，包括： 

a 旅行的行程距离 
b 旅行的季节 
c 目的地的地形、气候条件 
d 在旅行途中将参加的活动 

Step2 决定背包的基本类型 

软背背包（Soft Packs/Day Packs）： 

此类背包大小基本为中、小型，而且没有任何内外支架，没有良好的背负系统，就象你逛
街时背的那种。 

优点： 
a 比大背包更为便宜，而且方便使用 
b 比较轻便，体积较小 

缺点： 
a 对长距离来说，容量太小 
b 较为简单的背负系统无法承受较大的负荷，背起来不舒适 
外支架背包（External Frame Backpacks）： 

这类背包拥有非常牢固的外部支架，在背包和你的肩膀和臀部之间起着良好的支撑作用。
更均匀的在你的肩膀和臀部之间分担重量，使背负更为舒适。背包和背负系统（可调节肩
带、腰带、胸带等）均固定在背包的框架上。 

优点： 
a 在平坦或难度适中的路途中行走较为舒适 
b 背包内部容量较大，多数行李可放于包内。并有间隔和侧包，使得物品的归类存放比较
方便 
c 比起内支架背包，在身体和背包之间有更大的空间，使得身体和背包之间有更良好的空
气对流，保持背负时更凉爽、舒适。 
d 比同样容量的内支架背包来得便宜 

缺点: 
a 比同样容量的内支架背包体积要大，而且更笨重 
b 当包满载时，外支架的背包重心更高，在路况差的情况下较为危险。尤其是在需要攀登
或是滑雪的路上 

内支架背包（Internal Frame Backpacks）： 

这类背包将支架移至背包内部，使支架和背包更紧密的融为一体。同时也更好的将重量在
你肩膀和臀部之间进行分配。 

优点： 
a 背包更舒适的紧贴身体，使重心更稳定 
b 比背负外支架背包活动更灵活，更容易掌握平衡 
c 更为先进的外型设计，使得活动更加自如，特别是在穿越狭小空间时所发挥的优势 
d 背负非常舒适，尤其是那些拥有可调节背负系统的背包 

缺点： 
a 大多数内支架背包内只有1--2个分隔层，而且很少有侧包，使得物品的分类放置比较麻
烦（但现在很多背包可以添加外侧包） 
b 由于存放空间较为狭长，使得放东西更要仔细。不过只要细心、合理的使用，背包寿命
会很长，而且越旧越舒适。 
c 由于背包同身体结合较为紧密，所以会比较热，空气不易流通。但现在很多厂家在背负
系统上使用了很多高科技的合成材料，使得这一问题得到较好的解决 。 
d 在背包的开支上会比外支架包高一些。目前，背包的支架的材料常使用用铝合金，有些
高档的背包甚至会采用钛合金作为原料，当然她们价格也是相当可观的了。而支架的形状
大多为长条状，但也有些较高级的背包采用管状设计。 

带侧包的登山包 （Climbing Pack）两类： 



一类是体积在50--80升之间的大背包；另一类是体积在20--35升之间的小背包，也称"突击
包"。 
大登山包主要用于登山中运输登山物资，小登山包一般用于高海拔攀登或突击顶峰。 
登山专用背包是为应付极端环境的，制作考究而独特，一般包体瘦长，包的背部按人体自
然曲线而设计，使包体紧贴人的背，以减轻背带对双肩的压力。这类包都经过防水处理，
即使在大雨中也不会漏水。此外，登山包除用于登山，在其它探险运动（如漂流、穿越沙
漠等）和长途旅行中也被广泛使用。 

Step3 决定背包的容量 

最理想的选择当然就是有一个既能装满你所有装备，你又能够背负得动的背包（当然，不
是只能背一下的，而是要能背上好几天的）。不要选择过大的背包，虽然它可以装更多的
东西。 

容积： 背包的容积常以"升"（L）为单位。而根据用途不同，容积可从30升（日用型）到
80升（长途探险型）不等。大部分背包族会选用50-60升容积的背包，因为它足以应付一般
旅游探险的需要。 

一些需注意的因素： 
a 睡袋通常是置于背包内部，它往往需要15升左右的空间。而使用外支架背包时，可将睡
袋捆扎于背包外，但应做好防水工作。 
b 不同生产厂商所标示的背包容积的标准也往往会有不同。同样是50升的标示，不同厂家
生产的包会有一些大小上的差别； 
c 由于背包者的个人条件不同（如身高，体力等因素），也会影响所需选择的背包的类型
。所以，在选择背包时最好询问一下有关方面的专家（比如户外用品店的营业员）。 

Step4 关于背包的设计 



一旦你决定了你所要的背包的类型和容积，那么下一步就是比较一下不同背包在设计上和
独特功能上的不同。 

可考虑以下几点要素： 
易用性： 你当然需要一个容易使用的背包。装、取和打开、关闭都应非常简单。而这些往
往取决于： 
a 背包主体内的间隔数 
b 间隔的设计--前后分割、上下分割，还是两者的结合 
c 口袋的数量和位置 

功能性： 目前，很多厂家生产的背包都有容积调节功能，使你能更好的使用你的背包。可
能一个标示为50升的背包经过调整便可达到60升的容积。此类背包大都会在标示上有50L+
或50L+10的字样。这样的设计应该包含：更长的捆扎带（可紧固包内较小物件）；可扩充
的包体（应付在途中增多的装备）；外挂点（可将更多的装备置于包外）；可自由脱卸的
小包（日用包）。 

耐用性： 谁都希望一个包可以用上一辈子，除非是式样过时，不想用她。那么，背包的耐
用性便是你应该考虑的另一个重要因素。尽量关注一下背包的用料是否是采用了耐用的材
料，特别是拉练边的针脚是否牢固，是否有加固了的背带、背带扣和包底。在这方面最好
是选用名牌厂家的产品。 

缝合强度：包括肩带腰带和包体的缝合，包盖和包体的缝合，提环和包体的缝合，面料之
间的缝合、外挂带和包体的缝合等。（关键部位的缝合线一般是多道，多重保险）。 

配件质量：配件包括腰扣、所有拉练、肩带和胸带扣件、包盖和包体扣件、外挂带扣件等
。（扣件是反复使用的部件，要耐疲劳，低温下不脆不裂，结实可靠）。 

面料选择：包括耐磨程度、防水科技程度、面料轻重程度、防撕程度、柔软程度及防UV程
度等。（防撕布料多为网格形；一般的背包不具有防水功能，需要配防雨罩；耐磨的材料
多用在背包底部，接触地面和山石最多的地方）。背包使用面料的HD数也是衡量背包好坏
的重要指数。HD数表示了这种面料编织的紧密度，一般的背包采用的面料大都在300--400
HD左右，而较高档的则采用500HD以上，甚至1000HD的面料。目前，较高档的背包会采用D
u PONT公司的CORDURA面料，它是一种强度极高的防撕裂产品。很多厂家会用CORDURA做包
底，而更有用CORDURA制作整个背包的（如BIG PACK的PERFORMIC系列中的几款）。 

防水性： 出行难免碰到下雨或别人往你包上浇水，那么一个好背包当然也应该有较强的防
水性能。虽然你可在包上罩上防雨套或穿上大到足以盖上你背包的雨衣，但难免你的包还
是会遇到弄湿背包的问题。防水性不光和面料的HD数有关（HD数越高，防水性越好），更
重要的是面料内的防水涂层。现在市面上流行的涂料有PV和PU两种，PU的性能要比PV来得
好，当然也会贵一些，但总比倒出一包水来要好得多。 

Step5 确信她适合你 

背负的舒适度是选择背包最重要的因素。 

在挑选时最好是背负一个满载的包，而不是只背一个空包看看式样。使背包的背面尽量贴
近你的背部，那么背包将成为你身体的一部分。走上几步，看看是否舒适。选几个不同的
背包试试，并从售货员那里学会如何调节背带、腰带，为自己挑选一个最舒适的背包。 


应注意： 
a 腰带--背负系统中最重要的一部分。它能将大部分的重量传送到你的臀部（确切地说，
应该是骨盆），减少你的肩膀的负担，使你更感舒适； 
b 背带--背负系统中的另一个重要部分。一个优质的背带能使背负柔软、舒适，而且调节
方便； 
c 背负行程--并不是你背包走路的行程，这是指从背带最上部到腰带上部的距离。这应该
是一个可根据个人身高条件不同而进行调节的行程。很多背包厂商在这方面都有一些专利
产品，如BIG PACK的T.C.S.（TUBULAR CARRYING SYSTEM）。

Step6关于品牌

　 背包的品牌众多，如香港的NIKKO（日高），日本的ZZRO POINT，德国的BIG PACK 和V
AUDE（沃德）等。由于NIKKO较早进入国内市场，所以名气较响，价格也较高。BIG PACK背
包大多采用“V”型内架设计，完全符合人体结构和力学原理。VAUDE牌背包由德国VAUDE公
司出品，该公司是欧洲最大的一家也是最著名的一家登山、探险等户外运动器材的生产商
。该公司生产的VAUDE牌背包也以设计合理，种类全，质量好而深受各国登山，探险等户外
运动者的喜爱。奥索卡（OZARK）也是一个著名的品牌，其系列登山包中的珠峰型背包就是
典型的大型背包，MASK和ICEPICK属于中型背包。

Step7关于适用

容积在50升以上的大型背包，在放物品时，要把不怕磕碰的重物放在下部，放好后最好是
背包可以独自站立，如果重物较多则把重物均匀放在包内并且贴近身体一侧，这样整体重
心不会后坠。 

　　背包上肩有技巧，把背包放在一定高度，双肩进入肩带，身体前倾，靠双腿站立起来
，这是比较方便的办法。如果在无高处可放的地方，用双手将背包提起，放在一膝盖上，
面对背带，一手控制包，另一手抓肩带迅速转体，使一臂进入肩带，然后另一臂进入。 


　　背好包后，要将腰带拉紧，使胯部受最重的力，要将胸带扣上，拉紧，使背包没有后
坠感，行走时双手拉着肩带与背包之间的调节带，身体略前倾，这样行走时重力实际上是
在腰、胯部位，背部无压迫感，遇到突发事件，上肢可灵活处之。在无保护通过急流、陡
峭地段时，应放松肩带，打开腰带和胸带，以便万一出现危险时，能以最快的速度使人包
分离
