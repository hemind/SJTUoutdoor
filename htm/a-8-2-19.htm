:   [1] 攀爬者：安全带一个、大锁两把（攀爬右侧屋檐无需大锁）、粉袋一个
:   [2] 保护员：安全带一个、ATC或8字环一个、大锁一把、手套一双
: 2 Double check包括哪些内容？并写出double check的至少四个不同理由。
:   [1] 攀爬者应检查保护员：
:     1. 安全带松紧是否合适，腰带一拳，腿带四个手指
:     2. 安全带是否穿拧，安全带腰带、腿带是否正确反穿，且余出至少8cm
:     3. 是否正确安装ATC或8字环保护器
:     4. 安装保护器的大锁丝扣是否拧紧，大锁大头朝上，确保主绳远离锁门
:     5. 是否带手套
:     6. 保护员若长发，则要将头发盘起不能触及保护器，确保衣物不会卡入保护器
:   [2] 保护员应检查攀爬者：
:     1. 是否不超过自身体重的1.5倍
:     2. 安全带松紧是否合适，腰带一拳，腿带四个手指
:     3. 安全带是否穿拧，安全带腰带、腿带是否正确反穿，且余出至少8cm
:     4. 八字结是否正确、整齐，绳耳不能太长，双防脱是否打好，绳尾余出至少8cm
:     5. （攀爬左侧和中间道）两把大锁是否锁门相对地连接八字结和安全带速降环..
