——————————————————————————————————————
穿LASPORTIVA攀岩鞋很多年了，非常信任LASPORTIVA的表现，甚至有点依赖的感觉。
借此机会，跟大家分享我穿着这些攀岩鞋的感受和一些认识，希望能对大家认识LASPORTI
VA这个品牌，或者选购攀岩鞋都些帮助。
说得不对之处请专业人士点评。
今天将要测评的我现在手上的5双些，分别是Fatura Python Testarossa Solution Miura
 VS
先上全家福。

1.科技
为什么要先讲科技，我认为LASPORTIVA攀岩鞋的完美表现很大程度上来自科技的运用。
1.P3 system
这五双鞋里，除了Python，都使用了LASPORTIVA的专利 P3系统，简单的说，P3系统就是保
持攀岩鞋鞋型的技术，使得攀岩鞋在多年的使用后不会变形，表现不会减低。
P3是如何实现的，以前我一直认为是在鞋里加了一层底，P3的鞋感觉上前脚掌都比较硬，
但是穿过有P3系统的fatura之后，前脚掌也很软。现在我感觉P3应该是通过整体实现的。
总的老说，我觉得P3技术很大程度的保证了攀岩鞋在使用的寿命里保持好的表现。

2. No edge
No edge  技术使用在了speedster和 Futura上，在正真开始穿futura之前，我也曾怀疑过
这个圆头的攀岩鞋是否能踩住小脚点。
事实证明，我根本不用担心因为圆的鞋尖导致了踩不住小脚点，（我不能断言是因为鞋导
致了踩点更稳，因为90％还是来自你的脚法。）
同时，no edge技术不止是这样，我的确能体会到futura超好的敏感性以及非常舒适的脚感
，在No Edge技术推出之前，我们一直穿着有很尖鞋尖的攀岩鞋，可能需要时间来适应no 
edg技术。

同时，如果你比较solution和fututa，你可以发现不光是鞋尖，就连整个鞋的前脚掌的边
沿都是圆滑的。这也是No Edge的一部分，我的确能体会到他给我带来的穿着舒适感。

橡胶
橡胶是鞋的生命，现在LASPORTIVA 的攀岩鞋全部使用两种橡胶，Vibram的XS EDGE，和XS
 GRIP2。之前LASPORTIVA还使用过XS GRIP。以下这个表可以看出这三种橡胶的区别。同时
，橡胶也可以检验攀岩鞋的出厂时间，如果你现在还买到XS GRIP的橡胶的鞋，那肯定是存
货，而存放时间长的橡胶是会老化的。

这五双鞋，只有MIURA VS使用的是XS EDGE，其他四双鞋都用的是XS GRIP的橡胶。当然，
橡胶厚度各不相同。
Solution 使用的是3.5mm的XS  GRIP2橡胶，FUTURA使用的是3mm的XS GRIP2橡胶，PYTHON
使用的是 3.5mm的XS GRIP2橡胶，TESTAROSSA使用的是3.5mm的XS GRIP2
MIURA VS使用的是 4mm的XS EDGE橡胶，
不同的鞋使用不同厚度的橡胶，脚感和耐磨程度都不同，我认为耐磨程度是和橡胶厚度成
正比的。而鞋的脚感，也就是我们平时所说的鞋硬还是软，并不和橡胶的厚度有直接关系
，是一种综合因素造成的。

同时，橡胶的不同使用也导致了鞋尖的不同感觉，我个人把这两种橡胶鞋尖的感受归纳为
，面和点。
顾名思义，XS EDGE的鞋给人的感觉就是面，鞋尖感觉有一条边沿，适合踩小边沿。个人感
觉更适合攀登SLAB。
XS GRIP的鞋给人的感觉就是一个点。穿着的感觉明显有突出的一个尖点，适合踩小而突出
的点。
当然，橡胶还有一些其他指标，比如说粘性就是大家很关注的，目前世界上其他攀岩鞋生
产商，很多都使用vibram的橡胶， 510和evolve还有些其他品牌使用自己的橡胶。
岩友有的时候老是抱怨自己的攀岩鞋滑，但其实是自己的脚法或者核心力量出了问题。我
曾看见过有的岩友把新的solution整个底resole成了510的胶，完成丧失了solution鞋型的
优势，大家觉得有必要么？

鞋型
我把这5双鞋的鞋型归位两类，其实跟按照橡胶分类一样， FUTURA， PYTHON， Solution
  和Testarossa 分为一类，简单的说，这几双鞋都给我感觉很尖。
同时 Solution,Testarossa和PYTHON 在鞋尖大脚趾处有明显的弧度，感觉是大脚趾被顶了
出来，不会感觉不适，反而被动的用脚尖踩点，这样对新人有很好的好处，有助养成良好
的踩点习惯。

而FUTURA给人的脚感不太一样，我认为是介于点和面之间的简单，使用了NO EDGE技术，导
致FUTURA外侧踩点的范围更大，同时因为橡胶比较薄，FUTURA的鞋尖非常敏感，再加上增
大的外侧踩点范围提供了面的感觉，所以FATURA的鞋尖表现个人认为是最好的。
PYTHON鞋尖的感觉是介于solution和cobra之间的，软硬程度略比TESTAROSSA软些，但是尖
的程度反而会比testarossa高。更适合大仰角线路的攀登。
而Solution绝对是公认的攀岩鞋中跨时代的产品，完美的鞋尖，穿着的感觉是大脚趾被被
动的顶了出来，使得踩点更准确和敏感。

而MIURA VS给人完成不同的感觉。从侧面图比较可以看出，MIURA VS的弓度是这5双鞋里最
小的，同时他的前脚掌也几乎呈一平面。

他的脚尖的感觉是突出尖的面积比较大，从而形成了一条棱。非常适合踩那些没有明显突
出的点，而最重要的是，非常舒适，是我平时最喜欢穿的鞋。

包裹系统
我不太清楚这样称呼他是否正确，但的确也是我认为的攀岩鞋比较重要的部分，关乎攀岩
鞋如何紧紧的包在你的脚上。
FUTURA和Solution 使用了单个搭扣，多层弯折的包裹方式。
值得一提的是，FUTURA比Solution多了一次弯折，这就使futura在收紧搭扣之后提高了包
裹性。明显感觉 FUTURA的这个搭扣更起作用了。

PYTHON  的包裹方式，我更人认为是改良的拖鞋式，也是在cobra的基础上改良的。之前我
们看一些竞技运动员穿着cobra，总爱在中间位置加上胶带，以增加包裹性，所以后来的p
ython就在这个位置加了一条搭扣，使得在拖鞋式的攀岩鞋有了更好的包裹性。

testarossa使用的系带传统的包裹系统，优缺点明显，明显优点是能更大范围的调节包裹
程度，缺点是，如果调节的不好，反而发挥不出鞋的优势以及穿脱麻烦。

Miura VS的包裹方式是搭扣式的，但是不同的是，他使用了三层搭扣，比传统的两层搭扣
，三层提供了更好的包裹性。同时也使穿脱更加方便。

还有一点值得一提，比较futura和solution，有一个明显的区别，futura供脚穿入的开口
明显增大了很多，我个人感觉这样的设计改良使得这双鞋比solution更加容易穿入。而且
增加了舒适性。

后跟
岩友们往往很注重攀岩鞋穿入后是否能紧贴脚后跟，这有的时候成为大家选购攀岩鞋的一
大标准。但是每个人的脚型是不一样的。不同的鞋对于不同的脚的穿着感受也是不同的，
适合我的不一定适合大家。
同时，有的时候岩友做挂脚动作时觉得挂不住，反而抱怨鞋的后跟不好，我认为主要还是
没有掌握挂脚的技术动作，或者选购了不合适的尺码的攀岩鞋。
总的来说，LASPORTIVA的攀岩鞋相比510的是比较适合亚洲人的脚型的，亚洲人普遍跟腱短
，后跟圆。
这5双鞋的后跟照片：

比较可以发现，这5双鞋每双鞋的后跟都不尽相同。 
Solution的脚后跟被厚厚的橡胶完全包裹，这样完全没有开胶的可能性，但是也丧失了些
挂脚的敏感，由于橡胶比较厚，柔软度比较差，很难和后跟完全贴合。对于我来说，Solu
tion的后跟是这5双鞋里最不好的。但是之前跟定国交流，他却认为solution的脚后跟对他
来说是最好的，可见不同的人对不同的鞋的感受也不同的，但更重要的原因是因为他有很
好的挂脚技术。
Testarossa正好相反，它的后跟是这鞋鞋里最软的，如果你有一个很圆而大的后跟，那么
testarossa是你最好的选择，因为testorssa后跟皮革的延展性可以保证你的后跟充分的被
包裹。但如果你的脚后跟不那么圆，象我一样，那么就会出现一些空间。但是testarossa
后跟的敏感程度绝对是最佳的。
Futura, Miura VS及 python的后跟很类似，尤其是Futura（右）和Miura VS（左）

对于我来说，这两双鞋的后跟非常适合我，我的后跟几乎使完全贴合的。
而python的后跟，我认识是在前两双鞋的基础上改进的，比较miura vs和python可以看出
python在后跟的外侧多加了块包裹的橡胶（另一侧没有），这一侧正好的挂脚最多使用的
地方，从而增加摩擦面积。python的脚后跟对我来说几乎使完美，以至于我很难脱下来，
因为脚后跟和鞋之间没有任何空气，鞋完全吸附在了后跟上。很多岩友跟我有同样的感受
。

尺码问题。
尺码更是因人而异，说尺码之前先给大家看看我的脚

绝对的反面教材，多年来一直穿太小的攀岩鞋导致每个脚趾都被磨出很厚的老茧，所以我
不推荐大家穿得太小。
些鞋之间的尺码关系。以solution为基准。
如果你穿38的solution。我认为miura vs可以跟solution同样尺码，python和testarossa
可以比solution小半码，Futura可以比solution小0.5－1码。
我的尺码（仅供参考）：LASPORTIVA 的越野跑鞋 42－42.5 solution 38 python 37.5 t
estarossa 37.5 miura vs 38 Futura 37－37.5

Break in （撑鞋）
这是一个非常痛苦的过程，我这里将这5双鞋的撑鞋难易程度排了个序，供大家参考。这也
solution：最难撑，通常需要2－3周才能完成和脚。
testarossa和python：几乎相同，但是由于testarossa可以依靠鞋带调节，会比较好撑。
通常需要1－2周完全和脚。
miura vs ：比较好撑，因为miura vs有很大的开口，导致穿脱非常方便，撑鞋也就比较容
易。 1周左右就完全和脚了。
Futura：几乎不用撑鞋，第一次穿比较紧，但是第二次就完成合适了，非常出乎我的意料
。
LASPORTIVA的保养意见：
一些岩友在爬完之后就把攀岩鞋放在包里或者锁在箱子里，尤其是脚汗比较多岩友，攀爬
的过程中，鞋的皮革吸收了很多的脚汗，如果再把它们存放在不通风或者闷热的地方（如
车的后备箱），那么鞋很快就会开胶。尤其是皮革较软的鞋，如：testarossa和futura，
攀爬完一定要存放在通风干燥的环境下。这样才能保证攀岩鞋保持良好鞋型，不开胶。
另外很多岩友使用吸异味的活性炭，这在某种程度上也能保持鞋的干燥。但是我觉得相比
其他品牌的鞋，LASPORTIVA的鞋没有什么异味，至少对我这个脚汗很多的人来说，而我之
前穿的其他牌子的鞋真是臭的不能近身。
我看到的很多岩友都是非常珍惜自己的鞋子，做的比我好多了，好好对它们吧。
最后要说的是缺点：
每双LASPORTIVA的攀岩鞋或多或少都有一些小毛病，总结起来可以简单说LASPORTIVA的最
大问题就是不够耐用（durable）
下面我将这些鞋在我身上发生的一些毛病总结一下，我觉得很多问题是值得LASPORTIVA关
注的，请大家注意：
solution：搭扣的带子被拉断，每双都断，是我太用力了？
testoassa：鞋带几乎要断，我也见过已经断的。也是我拉鞋带太用力了？
python：至今没有发生问题。
miura vs：搭扣扣件断裂，还是我太用力了
Futura：搭扣的带子被拉断，后跟开胶，开胶我认为是我不注意存放环境造成的。带子断
裂是还是我太用力了。

总的看来，我是个暴力狂，但是我认为这些位置的断裂我认为是可以靠一些设计上的改良
可以改进的。
比如solution和futura搭扣带子的断裂，很多岩友都遇到过这个问题。
世界上没有穿不坏的攀岩鞋，不同的使用环境导致不同的问题，耐用和轻便，表现本来就
是矛盾的。希望以后能看到更全面的设计。

写到这里，基本上算是道出了我对这几双鞋的一些基本认识。
还是那句话：脚法和技术是最重要的，攀岩鞋不会把你带到5.13, 看看大魏同学穿着什么
样的鞋完成了闪电，你就知道脚法和技术的重要性远在你穿什么鞋之上。
但是我们还是很喜欢这些攀岩鞋，尝试穿不同的鞋攀爬也给我们带来了很大的乐趣，不是
吗？
希望以上的这些文字能对正在徘徊选择LASPORTIVA攀岩鞋的岩友有些参考。
穿着你自己喜欢的鞋享受攀登的乐趣吧。
