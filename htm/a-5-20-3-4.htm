: 一、【领队焦非庸部分】
: 国庆七尖是我领的第一条线路，虽总体问题不大，没有什么危险，但是还是有很多非..
: 得改进的地方，下面我将一一说明。
: （一）【前期准备】
:           确定领七尖这条线路大概是9月5号到10号之间的样子，当时我接下这个任..
: 是十分兴奋却又感觉任务艰巨异常的，兴奋是正常的，关于艰巨的话得从领队培训说..
: ~
: 当时领队培训的时候具体分配给我的任务是联系向导，然后在出发的车上担任领
: 队，在司机开车过程中注意线路等，实际上我所做的并非准备一条线路最重要的工作..
: 就导致了我在准备线路过程前着实感到任务艰巨，怕自己做得不够好。
: 而在着手准备之前，我甚至不知如何去开始，但好在有各路大哥的多方支持，我才得..
: 做好前期的准备（实际上算不上好，最多算良好）。
: 真的万事开头难，当我一直在考虑什么时候开始去准备，要多久才能完成的时候，我..
: 里非常浮躁，甚至有时候会有挫败感，也会觉得有一丝后悔——我靠，我为啥要去领..
: ，闲的没事干吗？自己跟他们去不是一样吗？何必要花费一个月的周期去准备，最终..
: 为那三天出线呢？
: 但是当我真正开始的时候，我觉得还行，还挺有意思，总体来说还是挺享受这个过程..
