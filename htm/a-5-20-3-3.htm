2018国庆七尖领队总结
（拖到现在实在不好意思，回来就一直赶作业，然后再准备新生杯和上海市攀岩赛，所以
耽搁了，请原谅我这辣鸡的办事效率以及令人着急的智商。。。。。。）

一、【领队焦非庸部分】
国庆七尖是我领的第一条线路，虽总体问题不大，没有什么危险，但是还是有很多非常值
得改进的地方，下面我将一一说明。

（一）【前期准备】
          确定领七尖这条线路大概是9月5号到10号之间的样子，当时我接下这个任务时
是十分兴奋却又感觉任务艰巨异常的，兴奋是正常的，关于艰巨的话得从领队培训说起了
~
当时领队培训的时候具体分配给我的任务是联系向导，然后在出发的车上担任领
队，在司机开车过程中注意线路等，实际上我所做的并非准备一条线路最重要的工作，这
就导致了我在准备线路过程前着实感到任务艰巨，怕自己做得不够好。
而在着手准备之前，我甚至不知如何去开始，但好在有各路大哥的多方支持，我才得以能
做好前期的准备（实际上算不上好，最多算良好）。
真的万事开头难，当我一直在考虑什么时候开始去准备，要多久才能完成的时候，我的心
里非常浮躁，甚至有时候会有挫败感，也会觉得有一丝后悔——我靠，我为啥要去领线啊
，闲的没事干吗？自己跟他们去不是一样吗？何必要花费一个月的周期去准备，最终也只
为那三天出线呢？
但是当我真正开始的时候，我觉得还行，还挺有意思，总体来说还是挺享受这个过程的，
特别是在看以往七尖的总结以及在看路线、做路线的时候——也是很久没有享受到那种全
情投入做一件事情的美妙感觉了！
首先感谢老铁啦（今年清明的时候走过一次七尖，老铁是领队，当时我和张大佬做他的副
领队），我很不耻的向老铁要了上次七尖的全套资料，并且也在之后的很长一段时间内都
叨扰了他，再次感谢！然后在拿到资料的时候emmm我就在想：“哈哈哈，时间也隔得不长
，而且我还走过一遍，干脆直接把老铁的copy一份给这次用咯!”然后还问了下几个人，说
是问题不大。但是有这个想法的我着实心里也是十分不踏实，想了想要是这样的话，我对
路线是真的可以说没什么了解了，虽是上半年走过一次，但是没有经过更精细的准备，我
敢再上去一次吗？要是我一个人，我肯定是说干吧！但实际上，我要做的是召集30余号人
并且将他们带上山并且最终将之一毛钱不差地还给拥有着他们的人。所以我还是怂了，乖
乖地仔细把会员线出线流程从头到尾看了几遍，并且实际操作了好几番地图制作过程（因
为之前对Google earth和global mapper的操作确实不熟悉，而经过了我那好几天的研究，
现已经算是比较熟悉了），最终完成了线路方面地准备，遇到了下面几方面的问题：
{
1.我再也不直接在Google earth上做地图了！
当时我记得花了大概一两天的时间在Google earth上点好了所有的点，删除了多余的路线
，做好了一切准备，就在准备转到global mapper上去制作路线的时候，我发现Google ea
rth根本不能导出我所需要的正确的航线。。。用global mapper打开的时候不是只有一个
点就是缺掉相当大的一部分航线！直到换届大会那天晚上我还一直再搞这个事，即便wrq和
sqc都来帮我看了看，但是都没有尝试好，当我回去了还是没有搞好，无奈只好照着Googl
e earth，再在global mapper上重新做一遍了，重复劳动实在是太多太多了。
2.在global mapper上做出地图图片的时候总是搞不好清晰度之类的，弄出来的图片要么小
的看不清，要么尺寸对了但是内存太大，就那种你不放大也看不出什么东西的图片。。最
后也是特地跑到光彪楼去找wrq帮助，好在都顺利解决，只是确实增加了太多重复劳动。但
是这时间花得值，让我印象深刻，下次出线准备肯定就不会如这番麻烦了。
3.队医问题。
    当时在找队医的时候简直费尽千辛万苦，直接大群里喊，没人理我。找医疗队长，没
办法，确实缺人，这就到队伍建设了，我就不多说了。找hhy学姐，结果她正好脚受伤了。
找学姐叫更高届的，结果也没了回音。所以最后只能压队兼任队医了，好在srt还是很靠谱
的，所以也有一个，我们的领压队也必须对医联知识更加熟悉！
4.领队组准备会开的时候可能没有足够明确线路重要节点。
当时开会的时候，我们确实领队组每个人到场，还有会长ldy(当时wrq说是要来着，但是我
忘了叫他，实在不好意思；而当时也可能没有提前通知到位，所以sqc忙于学业没有看到我
的提前大概半天还是一晚上的通知，也没有到场），总体开的还行，但是关于营地，我和
压队仅仅只是在地图上“统一”了，实际上我们认为的营地偏差了大概100米，这也是导致
“营地事件”（后面再提）的一个重要因素。所以以后开会的时候一定要更加明确再明确
，确保领队组信息及意见一致且完备。
5.向导问题。
      野风说自己没空，因为老婆不让他出去，想留他在农庄帮忙，所以好了他师傅苍狼
，当时他说是安排好了，但现在觉得不太妥，因为我没有和这个苍狼直接联系到，万一他
忘了，或者发生其他什么，那这个向导就很麻烦了。
}
然后准备好之后当然要发推送召集并筛选人了，实际上问题并不少，具体问题以及我的想
法将在此一一罗列：
{
1.召集帖风格问题。
发召集帖的目的便是吸引大家来报名，来参加我们的活动，所以必定不能过分严肃；同时
我们也不是来者不具，必须要说明准备会以及体测之类的报名成功的必要条件和要遵守的
线路必要的安全规范之类，做好定位的同时吸引更多的人。
2.体测以及筛人问题。
          这次筛人遇到了一些麻烦，有两个人认为自己通过了体测肯定会被允许参加线
路，然后买好了一套装备结果被刷掉了。那2个人不服筛选结果，引发了一系列争执，甚至
他们还对于我们的筛人机制产生了怀疑，认为我们的人员就是内定了，当时也引起了很多
人去干涉，最终还是妥协了，让他们俩加入。这次算是解决了，但是这依旧是一个bug，我
们要完善机制而不是就每个实例单独解决。
3.线路后期宣传问题（先讲了）。
          我们后期总结推送并没有什么特色，也不够能说明一条线路的美，一次体验的
珍贵，也没能宣传出我们的风格、理念，好像还听说总结推送用过前几次照片的故事。。
      而且这次也有一个大问题，也是我们的疏忽，宣传的时候忘记发到新生群里，导致
整个队伍没有18级学生，当时完全没有意识自己的朋友圈和空间里小朋友尤其对户外有兴
趣的小朋友其实并不多。。。
      }
          所以就以上三个问题，我说一下自己的想法：首先，召集帖不该是死气沉沉的
，我们可以更皮一点，套路多一点，然后对于线路的普通介绍更多一点，同时把必须的规
矩立起来，醒目的提醒！比如准备会、体测和领装备时间（当时有较多人缺席）以及行军
纪律，出线规范等等，同去网信息收集其实也可以直接统计是否有时间，没有就直接踢吧
。那么，体测和筛人呢？我们有一个统一标准吗？只取决于每次线路具体的领压队？其实
我觉得这次压队筛人确实有点草率了。但是也没有办法，导致我们一没有办法给他们具体
被退的理由，二不能让他们清楚知道自己被刷的可能性，就可能发生像这次的事情，我想
我作为一个会员或者参加活动的人也绝对会感到非常不爽！我觉得线路人员初筛的时候可
以先筛掉那些没有时间参加体测、准备会、装备培训的人（同去网设置信息收集选项）以
及黑名单人员，然后酌情筛掉没有任何经验的大龄非会员（经验>年龄，看有没有经验更重
要，当然先在年级高的里面刷）和近期受伤以及有较重旧伤的人员（设置选项），通过的
去体测然后体测，我的想法是定一个男女每圈配速的同时定一个关门时间，领队组跟着配
速带，最后在终点等着并记录，长于关门时间的或者排名靠后的超过预定人数的刷掉！当
然要保证这种体测的实施，必须要一点小道具，也很简单——写有编号的纸，这样就解决
了不认识人的问题。然后整个筛人过程必须有依据并且有记录，发短信还是群发，但是有
人不服，我们也可以拿出证据来！以上是我的想法，说的差不多了，我的流水账就go on了
。

（二）【走线过程】
        队伍规定10月1日17点于庙门集合，但实际上有人迟到了甚至15分钟，虽说迟到做
俯卧撑，可我们毕竟没有强制力来保证实施，也没有办法使用合法的暴力，所以我觉得以
后发通知别说什么时候出发，直接先过来，不然很多人就会认为时间充足，不以为然，习
惯性迟到（当时问过，他们大都觉得6点才出发，一个小时根本没啥事可干）。这显然不利
于队伍的行军，因为毕竟户外徒步也是一个时间观念十分强的活动。
        出发车程很顺利，一点也不堵，只花了3.5小时就到了向导家了。车上强调了一下
时间的问题，再做了一下常规的介绍，便不多言。
        到向导家之后很快组织睡觉，但是由于第一次当领队激动又紧张，再加上初次体
验b400加蛋槽热得不行，以及相互呼应的打鼾声，我是真得很难入睡，所以只好又在脑子
里过了几遍路线及安排，最后终于还是睡去了。本来叫副领队定好了3点50的闹钟，结果不
出以外，我又在闹钟之前醒了，反正心里装着事“自然醒”就变得看起来如此不自然。。
。
        早饭啥的略掉，到达入山口时5点19分，带大家做了一下准备活动，当时有一个队
员的头灯落在了车上，我却立马打电话给计师傅，结果计师傅就调头了。当压队催我出发
的时候却因此被耽搁了，这里我检讨，确实第一次做领队，可能想去尽量做好，因此做出
了很没有水平的决定，舍本逐末（现在想想，当时可能把好领队的标准弄错了，领队不是
要去迎合每个人，让每个人感到很爽，我们不是保姆，我们要做的是把他们安全带回去才
能谈其他，所以头灯这种事完全没必要如此）。
        一出发就是一段陡上坡，我很无奈地必须压缩自己的行进速度，可还是没办法使
队伍不脱节，只能控制别隔太远。6点56，我们到达仙人顶，拍了波照后继续前进。
    9点25分组织部分成员冲顶龙王山，其他人原地休息，9点40到顶，很多人拍照，最后
吼了好几嗓子才集结了原路返回，10点回到原休息点准备出发。10点30在一个陡长下坡结
束后吃了午饭。后面的路途中不知为啥我后面玩起了成语接龙，我也接了几个。11点44到
达节理峰，之后的路上见到了野风(说老婆开恩放他出来玩，还带的是老婆那边的人走地，
还真是怕老婆啊）、一些驴友以及一些调皮地小孩子，12点半到达药王峰。13点17登顶千
亩峰（icer一行人绕了过去在下面等我们，给了他们一个对讲机，方便交流）大约一小时
之后下到底，休息片刻出发。15点50到达营地，扎营做饭，旁边有小卖部，我们购买了热
水给大家补充。但是水很烫，我们便将水倒入锅中，在将锅浸泡在装满冷水的大盆里进行
热交换，可效率也不是很高，搞完都快到睡觉时间9点了。第一天基本结束，白天有一个男
同学在下坡突然摔倒，真不知为什么，好在没受伤。
        第二天5点半起床，预定7点半出发，但有2个组磨叽耽误了10分钟才得以出发。一
路公路到山脚，然后再从小路抄上去，虽然难度不大，但队伍难免会有一些脱节，我们就
每次在小路与公路交点处等待。8点54分到达泥岭休息，9点02部分人从防火带上去冲顶桐
杭岗，这段没有岔路，队伍于10点06登顶，比时间节点晚了接近半小时，所以下山走公路
的时候催促大家快一点了，同时也将午餐点直接提前到了泥岭这里，最终12点继续出发。
14点38登顶仰天坪之后，发现下面居然都长了好高的草，变化较大。
之后就是营地了，本来3点50就到了我预定的营地——清明使用过的营地，但是当时我跟向
导说的时候，他说到有更好的营地只要下去5分钟，结果我选择了相信，并且觉得大家都提
前了十分钟，我们何不下去让大家有更好的驻营条件呢？结果带着队伍下去了，而到了那
个营地时（也是压队所认为的郜林头营地），发现已经长草了，当时向导又说继续向下还
有公路旁可以扎营，便继续下了此时我的对讲机也没电了，能听到压队，却无法发送信息
，这也导致了他犹豫之后还是跟随队伍下去了。然而公路旁的营地遍布石子，有点不方便
，然后再埋怨的时候，向导便说一直向下走，20分钟有人家，可以买东西，方便扎营，在
他电话联系好那边的人后，我们便又出发了，但实际上我们走到了4点40也没到那个地方，
压队就直接决定不往前走了（确实队伍状态也就那样了），就在2号楼前面扎营了，遇到房
东不同意，还多亏了压队的社交能力，我们才能不用被赶走，那晚大家8点睡了，但是当我
确认都安顿好了进了帐篷之后，又有人在讲话了，但是居民区不好直接吼，所以我在群里
发了消息叫他们安静。由于当时已经沿下撤路线走了有一段距离了，于是压队在凌晨醒了
的那段时间去翻六只脚，还找到了一条可以近很多的航迹，并决定在起床后一个小时时去
和副领队探路，然后在出发前20分钟左右回来了。
关于此呢，我想表达一下我当时的想法，以及之后我的看法。首先，当时我过分相信向导
了，按理说选定了营地就不应该轻易改变，不是说说什么，向导说有更好的营地我就相信
并且带着队伍下去；再者，和压队沟通不够，我们口中的营地甚至不是同一个地方，然后
当时到营地之后我确实应该在原地等待直到压队赶到之后再决定
——尤其是换营地的大决定（其实也没必要换），以及到后面营地长草也一样，也没有等
待压队，这是我的严重失误！
          然后关于探路，个人觉得必要性不大，首先我们的最终营地距离原定路线的入
山口（因为下到了公路）之间的距离没有超过一公里，再加上公路好走，再慢也不过半小
时，实际上我们用了差不多20分钟（这还是压队几次停来和向导讨论所谓“近路”的情况
）下，确定就到了入山口，而在原营地下来的话也绝对没有快很多，甚至我不觉得会更快
！再加上通过前两天的观察，这群人的体能确实还行，就算落个十几二十分钟，我也能让
他们赶起来。那当时我为什么会同意他们去探路呢？我想说：我不同意，他们会安心吗？
凌晨3点查了航迹，始终担心会非常晚到，或者中午下不了山，我也没办法啊。压队，副领
不安心，在队伍里会传达什么什么情绪？再者，我还是相信他们的能力的，手机有航迹，
不至于找不到路，他们体能都不错，不会挫。而且但是天已经比较亮了，危险不至于很大
，在可控范围之内，于是就没有强烈反对。
第三天总体很顺利，不过确实下坡带的有些快了（其实我觉得紧跟我的那组还跟得挺好的
啊）后面也放慢了速度。
向导家腐败有些坑，钱花了，分量不足，华而不实，以后绝对要要求分量！
车是真多！最后6点50到达庙门，平安归来！
