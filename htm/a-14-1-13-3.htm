标  题: Re: GORE公司相关物品保养
发信站: 饮水思源 (2005年10月15日01:41:54 星期六)
要补充一点：
不管是什么公司做的防水透气面料，最终都要通过一定的工艺，使面料薄膜贴合到其他面
料上去。
所以这类面料的保养方法大致都是类似的。
不喜欢GORE-TEX面料，或者觉得他贵的离谱又没有理由的朋友们，尽可以选择其他的防水
透气面料。
不管什么面料，但凡是防水透气的总还是有点贵的，大部分没GORE-TEX贵地这么离谱，可
还是好好保养，延长寿命比较划算。

还有一个问题要解释：
有很多朋友一直认为GORE-TEX面料有2年保质期，过期，就没有用了，那我还保养啥呀，反
正是要坏的。
其实，这里说的保质期，指的是GORE-TEX面料在‘生产出来之后到压胶到其他面料上之间
’的库存保质时间。这段时间超过2年，面料薄膜本质是塑料，会因为双面暴露在空气中而
迅速的老化，不适宜再压胶到其他面料上。
当GORE-TEX面料一旦压胶到其他面料上，他的寿命是相对比较长的，前提是正确的使用和
保养。所以保养很重要。
【 在 judieth 的大作中提到: 】
: 1）带GORE-TEX的服装（其他防水透气面料都一样处理）：
: 用40度以下的温水加适量洗衣粉机洗或手洗。
: 不要使用漂白剂或柔顺剂。
: 有条件的使用55度烘干机烘干，有助于恢复衣服表面的拨水功能。
: 可以低温蒸汽熨烫，可以达到类似烘干的效果。
: 可以偶尔干洗，但是提醒店员不可以用针刺穿衣服。
: PS：我的建议-最好手洗，减少损耗。尽量保证低温熨烫。
: 2）带GORE-TEX的鞋子：每次穿过出门后都要清洁保养
: 先用软刷（如牙刷）刷去泥土等脏东西，倒出鞋内的砂石，取出鞋垫，放松鞋带（视..
: 肮脏程度，决定是否拆下来清洗），自然阴干（如果非常潮湿，用报纸塞进鞋子内吸..
: 。
: 穿鞋子前，请确保脚趾甲剪干净了，否则GORE-TEX可能会被顶穿。
: 建议3个月左右做一次拨水处理（用专用喷剂）。
: 注：拨水功能可以使鞋子表面保持干燥。有的时候下雨鞋子表面湿了影响透气性，外..
: 气湿度又很大，脚上汗气肯定在皮肤表面凝结。这会让人误以为鞋子漏水了。
: 3）带GORE-TEX的手套：
: 采用机洗或手洗。
: 不要使用漂白剂或柔顺剂。
: 不能熨烫，不能干洗。
: 采用手套手指向上的方向晾干，不建议拧干或用洗衣机甩干。
