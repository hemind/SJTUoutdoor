: 16日出发前往北京，这次赶火车还是比较紧张的，7点53的动车，接近7点半的时候我..
: 火车站，还好取票和案件都没有什么人，还是很稳的上了火车，以后还是早点出发，..
: 不赶晚啊。老大哥更危险，大概再晚两分钟我就可以一个人去北京了。
: 17日早上到达北京，下车就感觉到北京是真的冷。我成功被之前那个在群里发被密云..
: 的人给误导了。之后就是挤地铁环节，北京地铁我是真的服，一分钟一班车还可以班..
: 载，大概4、5班车之后才成功挤上地铁，到东直门之后换980快到达密云，等到包车..
: 后，在物美买了中饭和一条秋裤之后前往云蒙峡。
: 17日中午到达云蒙峡，第一反应是肯德基是什么鬼，心情异常沉重的拿到了登山鞋，..
: 就是还装备去冰壁，刚换完就成功摔了一跤，不是很懂为什么要铺瓷砖，真的是好滑..
: 之后每天出门都特别小心。之后就是冰坡行走，反正就是复习吧，本来就一年只有一..
: 冬训还断了一次很多东西都忘记了，重新复习一下。感觉吧就是在那个每个人走一遍..
: 看的环节还是不错的，但是比较费时间，还有一个我觉得比较关键的地方就是在指出..
: 之后不会一直纠正到对为止，我觉得这样可能不太好。
: 之后几天就都是攀冰了，第一天从冰壁后面翻过去建了一个站，啊，选打冰锥的位置..
: 是好纠结啊，前两个锥很顺利，第三个就碰到冰碎了，冰里面空心，旁边有冰洞的问..
: 反正花了很长时间，这个建站选址水平0分。然后就是正常的爬了几次线，然后在整..
