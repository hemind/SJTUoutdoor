:                              观音尖感想
:     参加这次徒步，最初是想跟着老司机们熟悉一下野协的活动流程，掌握基本的生..
: 能，顺带弥补国庆没出去浪的损失，认识一堆志同道合的小伙伴。很幸运，这些想法..
: 现了，不虚此行。
:     从最开始的动员大会，到装备的分发、学习，我能感受到这是一个有组织的团体..
: 位老人都愿意将自己的经验分享给新人。第一次背上那么大的登山包，有种难言的欣..
: 未知的渴望，仿佛一下子在全新的领域跨出了第一步。
:     我虽然是四组组长，却并没有任何相关经验，幸好组里有麻磊这个老司机，大部..
: 作都是他组织的。周四去欧尚采购食品，才算是我与组员的第一次见面。大家都对徒..
: 满期待，相互讨论协商，共同完成了“吃”这一大事。也许是缺乏常识，购买的食品..
: 丰富但是偏重，对第一次徒步的人来说可能有些负担，也算吃一堑长一智。
:     我自己在打包的时候，装了很多不需要的东西，所幸是第一次，没有对比就没有..
: ，也倒不觉得有什么负担。坐车到达出发点，又因为无法扎营必须睡在车里，我想对..
: 数人来说都比较难以接受。因为有过做一天一夜硬座的经历，对此没有太大感觉，只..
: 半夜车里实在太过于闷热，总体来说休息得还可以。旅途中大家做自我介绍，玩破冰..
: ，很快气氛就变得融洽起来。
:     第二天一大早，小雨淅沥，出发。先是机耕路，小时候跟着奶奶走过，倒是勾起..
: 忆。然后是石头路，我穿的运动鞋，走上去不打滑，穿登山鞋的小伙伴却觉得有点难..
: 到了较为陡峭的地方，由于刚下过雨，土质松软，时不时有落石纷纷而下，煞是危险..
: 个姑娘不幸被落石砸到脚踝，却坚持到最后，令人敬佩，当然这都是后话。按照顺序..
