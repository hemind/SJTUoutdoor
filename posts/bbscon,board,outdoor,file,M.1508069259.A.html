: 清凉峰队医总结
: 一.线路难度：一般，第二天的下坡行进较为缓慢。
: 二.出线前准备：
: 队医的注意事项：
: 蚂蝗
: 1 裤腿，长袜，厚袜子
: 2 不要用手抓——科普一下蚂蝗的处理方式
: 晒伤
: 1防晒霜（每组有一小瓶就行）
: 磨脚：可以穿或者至少两层袜子，或者准备好贴在脚跟的创口贴之类的…新鞋一定要..
: 	✓	身体不适要及时喊我！！！！！
: 	✓	提前上报：1.药物过敏 2.晕车3.有无重大疾病
: 	◦	女生注意生理期。
: 	◦	自己有单独需要备的药物自己要记得带
: 	◦	出发前不要献血。
: 	◦	建议不要买太辣的东西，容易肠胃不适……具体还是根据各自的情况
: 	◦	护膝，护踝——膝盖和脚踝不适的话记得带。戴劳保手套，最好穿长袖
:       
: 记得剪好手指甲和脚趾甲。
: 出发前和出发后有身体不适都不要强撑，要及时提出来。
