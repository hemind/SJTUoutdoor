: 2017小学期衢州三十六湾压队总结
: 第一次当压队，心里很忐忑，对于自己具体应该做的事情没有太多经验，只能依葫芦..
: ，在两天行程里不断摸索。
: 这一次副领队是个妹子XYQ，刚得知这个消息挺出乎我的意料，毕竟之前出过的十几..
: 领都是汉子。几天下来，做事情干净利落不拖沓，值得表扬，只是委屈了她线路简单..
: 的作用力体现不大。领队SRT非常靠谱，在营地催大家睡觉一脸严肃凶残不留余地，..
: 奏把握得非常好，基本没有脱节情况发生。队医CYJ害怕昆虫，一路走下来能把大家..
: 蚂蝗都拔掉也挺不容易，一路受伤情况也很严重，创口贴基本发放完毕，主要是石头..
: ，鞋磨破脚，蚂蝗伤口等。
: 以下是线路过程中的一些问题和建议：
: 1、这次小学期间衢州天气晴好，直接住在向导家门口，水泥地，不能防雨，目测可..
: 最多15顶帐篷，晚上帐篷里十分闷热，第一天晚上12点半左右到达向导家，领队在帐..
: 打充气垫五分钟，中间我都不敢进帐篷，内外体感温差实在太大！进了帐篷辗转反侧..
: 小时迷迷糊糊入睡。建议以后队伍多教一些帐篷通风的知识，包括天窗和内帐上部的..
: 气窗（很多人不知道），以及如果天气不错可以考虑掀开部分外帐入睡。向导家在河..
: 晚上湿气很重，包直接放在外面早上起来全湿。
: 2、这次天气晴朗，蚂蝗依然严重，很多人被咬。途中遇到竹叶青，要注意。
: 3、这次溯溪只遇到一个难点，在水路三分之一处左右，水潭左侧水深大概170，右侧..
: 上，水流湍急。没有布绳，由几个会游泳的同学协助通过。水很冷，水刚淹没腰部时..
: 难受。其实应该布绳通过的，等我到达这个点的时候队伍已经过去快三分之一，索性..
