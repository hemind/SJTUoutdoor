换洗衣物还是要的，虽然说可以不带太多，但是一套干燥的衣物可以让你在面临失温的风
险时给你一个干燥的环境。在温度较低的地方，万一下了暴雨（冻雨），到营地在帐篷里
面换上干燥的衣物，特别是即将面临失温的队员，个人感觉是很有必要的

反正我自己在山上是不换衣服的，因为一般到营地衣服也就干了，有衣服也懒得换。出了
这么十几次线，备用的衣服用得上的也就是这次观音尖，在淋过暴雨后快冻得不行的时候
（背着包在走也冻得不行啊喂）换上一套干燥的衣服是一件多么爽的事情

至今记得鳌太山难时，别人关于失温的帖子里的这句话：在十摄氏度的大风天，人可能会
失温；在二十摄氏度的大雨天，人可能会失温（啊反正就是这么个意思）十度二十度很容
易达到，然而万一遇上大风或大雨（概率很小），那么如果有充足的准备，就可以从容应
对这些恶劣天气

: 装备总结
: 为了给今后的登山留下参考，也为了让大家能够重视装备的重要性，在这里对2016年..
: 交通大学极地登山队的装备使用情况进行简要总结，并分享一些感想和体会。在装备..
: 总表中，我总结了这一次登山携带、使用的装备，分为了个人装备和攀登公用装备，..
: 的队伍在准备时可以酌情参考。
: 一、	前期准备工作
: 1.	确定人员，明确大家的装备现状，着手准备装备计划。
: 2.	根据登山计划以及往年登山报告书，确定此次登山的装备清单。
: 3.	通过装备部部长询问协会装备库存状况，参考队员个人装备情况，确定装备缺少..
: 准备装备赞助，租赁，团购。
: 4.	同协会干事合作拉到黑冰，高特，Singing Rock，的团购赞助，以及极地，Fenix..
: 普奈特的全额赞助。
: 5.	协会内部统计团购数量，下单并发放团购商品。
: 6.	个人装备统计数量，发放，监督队员准备个人装备，监督队员清理装备。
: 7.	卫星电话租赁，保管。
: 8.	公共装备的整理及装包。
: 9.	所有电器充电，确定所有装备处于最佳状态，准备出发。

: 在成都留下部分个人生活用品。队员讨论之后，将徒步鞋与登山鞋也留在成都，准备..
