: 登（da）山（pai）日记
: 2016.8.13 
: 上山第一天，虽然上山前感觉状态非常不好，但徒步到海拔4000的营地却意外的轻松..
: 本说营地3900，信息有误），几乎没有高反症状。教练一直叫我们慢慢走，别出汗，..
: 一个能边走边聊的状态。杰哥说这才是徒步正确的打开方式，想起以前走线走得气喘..
: ，可能是强度超过了自己的能力。途中有人因早饭没吃好而呕吐，全部吐完补充能量..
: 恢复了一些。
: 期间实地了解了LNT原则在行走途中的一些应用，比如不要走在路两边、荒野的分散..
: 则等等，深感LNT并不仅仅是不留垃圾那么简单。到了营地，挖厕所、搭帐篷，不过..
: 于走在最后并没有参与这些。
: 2016.8.14
: 昨晚几乎没睡着，12:00钻进去，1:00多睡着，醒来听到涛哥起来喝水，说才2:30，..
: 迷迷糊糊尝试入睡，4:30左右醒了一次，之后几乎没再睡着过。7:30听到闹钟后起床。
: 早上起来并没有因为睡眠不足而感到累，精神反而特别好。
: 上午介绍了各种装备，高山靴的选择、穿法等等。
: 下午1:40左右穿着高山靴进行适应性行走，速度宜人。走到冰舌处欧阳老师给我们科..
: 冰川知识，随后继续爬升，4:10走到冰舌再往上一些的地方，看到一台挖掘机，此时..
: 了约500m，教练上山探路，我们原地休息，4:20返回营地做饭，5:10到达营地。
