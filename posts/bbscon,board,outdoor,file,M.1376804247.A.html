                          浙东大峡谷脑残游记

（心情不好的都来看看吧~~）
忽然想起游记没写，仔细一回想，记忆模糊了许多，吓了一跳，赶紧写下来。
首先对秒退党表示感谢，没有你们估计我等秒进不了。。。。好了，下面开始游记正文~~
28号晚上我悄无声息的迟到了两分钟，幸好没人发现（汗。。）。来到就听说王森忘带洞
洞鞋回去买了，那是个什么玩意？就是那其丑无比的鞋？想起我就随便拿了一双足球鞋，
顿时菊花一紧，然后就开始了不断的自我安慰。。。
上车后我一直很淡定的休息，一直到自我介绍时才发现脑残七侠到了六个，顿时觉得这次
一定会发生很多愉快的事情滴。。。
第一天晚上，12点多到达营地，又是个打不了地钉的营地，只能用石头压，但我发现用石
头很难保证内外帐分开（老人知道肿么弄的话说一下），想起自己带了个绳编的手链，拆
了3米多长，果断拆了一分四段，用下来感觉凑合，内外帐基本分开，于是乎我就开始祈祷
夜里下点雨（我的目的只是为了检验一下帐篷搭得怎么样。。。）。刚入睡一会，两点多
的时候，旦旦的队伍来了，他们的车停在我们帐篷旁边很久还没熄火，我心里那个纠结呀
……然后车上下来几个人，感觉比较吵，再后来他们好像去向导家了。
29号六点半起床，但发现才六点多点的时候就很多人起来了，T。T吃早餐时居然发现大脑
残他们组带了自热米饭，这……出发后一段平路然后一段爬升，再然后瑞叔说今天的爬升
结束了，顿时吓我一跳。接下来的下坡没什么好说的，就是瑞叔、皓皓老师和其他几个老
人在后面回味春晚的某个小品，一直听到他们在那喊诸如“漂！！！！亮！！！！”、“
亮相吧，小宝贝！！！”之类的话，然后就是各种无下限的卖萌。不久后到达溪水边休息
，开始换鞋。大脑残和四脑残在鸳鸯戏水，非把自己弄湿了才舒服，于是我默默的把一个
大石头仍在他们附近的水里~~三脑残那时好像在啃黄瓜。。随后开始换鞋，看见身边一双
双洞洞鞋，我默默拿出足球鞋，心里拔凉拔凉的，隐隐感觉这鞋不太靠谱。果然，刚开始
走了一会就滑了两次，虽然凭借我犀利的身法而没摔倒，感觉可以搞个诗意的名字，“当
足球鞋鞋钉遇上石头上的青苔”，肿么样，是不是觉得非常好听、非常牛逼？不过庆幸的
是后来自己慢慢适应了这鞋，也就没滑过。（我靠，我好牛逼，超神一般的存在呀！！）
11点多到达午饭地点，开始午饭，一边看着瑞叔秀美腿一边啃翔一样的面包。然后和大脑
残他们分食两个香瓜，感觉没上次金紫尖的甜。
之后一路就是各种涉水，没什么好说的，我就是跑前跑后，水流急、易滑的地方拉拉妹子
呀（当然汉子偶尔也拉T。T），然后就是岸边有些点得依次翻过岩石的地方，翻过去后绕
道水下用手给他们提供脚踩的点什么的。途中偶遇旦旦的队伍，数了一下他们前半只队伍
十六七人只有两根登山杖，这你丫能信？然后很快就到了本次行程最好玩的地方。是这样
的，我们走的岸边走不下去了，得横渡过去，然后奥叔、皓皓老师、领队等等开始四处寻
找最佳渡河路线，然后我也背着包下水了，去了我就后悔了，水很快就接近我的胸部，而
且背着包及其不好走，领队在我十点方向不远处，他那水早过胸部了，而且水底石头巨滑
无比，于是我退回去了，上岸后，伸手往后一摸，背包罩给我罩了一大兜水，这。。。把
背包罩底部拉开，“哗~~”那家伙，那场面，那叫一个壮观……很快就决定从一个渡河路
程较短但水流较急的地方渡河，于是乎我就很悲剧的被来去做人墙了T。T，拿杨佳佳实验
了可行后开始运包，随后渡人，虽然有人墙，但还是有人摔倒，而且还冲跑了两根登山杖
（不过后来在下游找回来一根……）对了，这里说一点，他们过河时总是拉我手掌，拉到
手腕上面才稳呀。。。由于这次渡河耽误很久，我们就又遇见旦旦的队伍了，他们选了我
们上游一个水很深的渡河点，为什么呢？旦旦的居然带了充气筏，这太捉鸡了，早知道让
大脑残把他寝室里的那个充气的玩意也带出了，╮(╯▽╰)╭哎，失策~~
渡过去后又是各种涉水、横渡，和之前差不多，随后再走了一段山路就到营地了，迅速搭
好帐篷后一群猥琐男立马跑去游泳了，水边有几个看客，以三脑残为首的禽兽们闲着无聊
，就用水把杨佳佳给泼湿了（这群丧尽天良的禽兽呀呀呀。。。。）。由于哦呜不会游泳
，我就默默在水里尝试用各种姿势学游泳，我发觉仰泳是个好东西呀，我每次用仰泳的姿
势时都能很快的沉下去，这……不久后，大家都上岸了，正当我默默的用头巾擦身上的水
时，某个人考虑到水温很低、大家都很冷，于是就全裸出镜，让大家兽血沸腾一下，是不
是呀，严皓老师？？（我不会告诉你们当时是一阵阵欢呼呀“巨大无比”“硕大无比”）
，好吧，我什么都没说。
一会开始做饭，我就拿了副碗筷四处游荡觅食，这组要点那组要点，要完一圈后回去，恰
好，我们组的第一锅好了。吃完又继续去游荡，不给？好办，绕两圈后回来道：“来，我
给你们组合个影。”“好呀。”拍完照，“你们的培根分我吃点吧~~”然后我就端着碗培
根边吃边寻找下一组。晚上篝火，外加各种无节操游戏，自行脑补吧。
篝火的时候，一二三四脑残我们开始煮我们的宵夜（纯属娱乐，不是为了吃。。。），先
说一下，大脑残和我额外多带了四个香瓜、一个菠萝、一袋八宝米、三袋纯牛奶（本来我
想要椰奶的）、两袋馒头、两袋方便面还有士力架，三脑残多带了五根黄瓜和四个苹果，
随后我们开始煮了，对了，插一句，中途，据老四说，三脑残王森带了根黄瓜上厕所去了
，这是为什么呢？（求解答求解答~~）言归正传，我们开始煮粥，我们先把八宝米用水煮
了一会，然后吧牛奶加进去，然后我们就一边吃水果一边把香瓜、菠萝、苹果、黄瓜都扔
进粥里去煮，巨香呀！！忽然想起我们没带糖，于是我们又塞了根士力架进去，我们真是
太聪明了，这种方法都能想到，人才呀！！很快，我们的苹果黄瓜菠萝香瓜士力架牛奶八
宝粥（又称十全大补粥）就好了，没人一碗，我勒个去，那味道、那家伙，那叫一个好吃
呀，唉，不说了，你们这群凡人是不会懂滴，嘿嘿~~唉哟，不说了，我的口水。。。。。
晚上十点就寝，可附近帐篷总是有人说话，领队怒吼两声才安静下来。是夜，天降大雨，
数帐进水，湿，难眠，数人携睡袋、防潮垫至屋檐下，乃眠！翌日，雨小，早餐，余组面
缺，无奈，四处讨之，于大脑残、四脑残、七脑残及杨佳佳组讨挂面，收获颇丰，立回煮
食之。好吧，我只能说，不会用文言文了，接下来拍合照、出发，依次过吊桥，。由于昨
晚大雨，决定走山路下撤，没有继续溯溪，这段山路难度真不小，感觉很昱岭关有的一比
，最后快到时，发现一条蛇，仔细观察后判定是菜花蛇，于是决定给它拍一套写真，拍了
一张后相机没电，我哭/(ㄒoㄒ)/~~。。。至于归途就没什么好说的了，最后发言依旧是各
种爆料、各种吐槽~~

:                           浙东大峡谷脑残游记
: （心情不好的都来看看吧~~）
: 忽然想起游记没写，仔细一回想，记忆模糊了许多，吓了一跳，赶紧写下来。
: 首先对秒退党表示感谢，没有你们估计我等秒进不了。。。。好了，下面开始游记正..
: 28号晚上我悄无声息的迟到了两分钟，幸好没人发现（汗。。）。来到就听说王森忘..
: 洞鞋回去买了，那是个什么玩意？就是那其丑无比的鞋？想起我就随便拿了一双足球..
: 顿时菊花一紧，然后就开始了不断的自我安慰。。。
: 上车后我一直很淡定的休息，一直到自我介绍时才发现脑残七侠到了六个，顿时觉得..
: 一定会发生很多愉快的事情滴。。。
: 第一天晚上，12点多到达营地，又是个打不了地钉的营地，只能用石头压，但我发现..
: 头很难保证内外帐分开（老人知道肿么弄的话说一下），想起自己带了个绳编的手链..
: 了3米多长，果断拆了一分四段，用下来感觉凑合，内外帐基本分开，于是乎我就开..
: 夜里下点雨（我的目的只是为了检验一下帐篷搭得怎么样。。。）。刚入睡一会，两..
: 的时候，旦旦的队伍来了，他们的车停在我们帐篷旁边很久还没熄火，我心里那个纠..
: ……然后车上下来几个人，感觉比较吵，再后来他们好像去向导家了。
: 29号六点半起床，但发现才六点多点的时候就很多人起来了，T。T吃早餐时居然发现..
: 残他们组带了自热米饭，这……出发后一段平路然后一段爬升，再然后瑞叔说今天的..
: 结束了，顿时吓我一跳。接下来的下坡没什么好说的，就是瑞叔、皓皓老师和其他几..
: 人在后面回味春晚的某个小品，一直听到他们在那喊诸如“漂！！！！亮！！！！”..
: 亮相吧，小宝贝！！！”之类的话，然后就是各种无下限的卖萌。不久后到达溪水边..
