然后关于锁紧装置，个人还是推荐旋转膨胀式的。
那种类似自行车快拆的东西其实从机械结构的角度来说不好
1.不稳定，容易开
2.单方向加紧调不好要变形
3.开关多次后凸轮有磨损会比较厉害

丝扣的那个最大的bug就是沙子，容易卡死，不过这么器械依旧采用丝扣最关键的还是稳定

: #####1.首先是为什么要用登山杖。#####
: 当然，随便一根木棍、竹子都能起到杖子的作用，但是这种作用是很局限的。
: 第一，自制手杖抓握不方便：没有手柄腕带，为了抓牢杖子，需要花费多余的前臂力..
: 久而久之容易疲劳，抓握不能（类似于登山杖使用不当的情况）。也同时因为需要靠..
: 牢杖子，在上下坡的过程中，不敢、也不能把过多的重量支撑在杖子上，使得登山杖..
: 力作用降低很多。
: 第二，长度调节不变：通常上坡手杖需短，可以省很多力；下坡手杖需长，提供额外..
: 条腿的支撑，平稳且保护膝盖；平路手杖长度适中，运用得当可以节省相当的体力。..
: 不是随便一根竹子、木棍所能替代的。
: 第三，杖尖提供稳定的支点。登山杖的杖尖一般有凹槽或者纹路，使得登山杖拄在各..
: 面，都能提供很好的支撑，不打滑，尤其是在湿滑的岩壁上，而用一根木棍拄在石头..
: 用力是很危险的事。
: 第四，携带方便，坚固耐用。这个就不多说了。并不是所有地方都能找到木棍和竹子..
: 没有人会随身携带这么一根木棍。
: 第五，还有很多其他好处，比如减震、搭天幕等，就不再展开了
: #####2. 然后是，为什么要买一根好的登山杖#####
: 诚然，登山杖是消耗品。之所以消耗，主要也是因为登山杖的横向受力很小，容易弯..
: 使用不当的话，非常容易坏。当时好坏登山杖之间的差别还是很明显的。
: 第一，好的登山杖的腕带很舒服。登山杖的使用主要是依靠手腕通过腕带向下压来提..
: 下的力，而手是不能握紧手柄的。这是腕带的好坏就直接关系到登山杖的使用，以及..
