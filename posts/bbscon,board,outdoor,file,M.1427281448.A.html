: 首先，这篇文章不是写怎么用GRIGRI下降或保护的，只是对错误使用GRIGRI可能出现..
: 些问题做一分析，不尽全面，欢迎大家补充。请在足够的学习后再进行相关操作。
: 在网上看到或听到有些人说GRIGRI是自锁保护器，或说具有自锁功能，我认为这是误..
: 有些人在体会了用GRIGRI下降，发现有时不用给制动端施力就能制动后以为在使用GR..
: 时可以放手制动端。其实在使用GRIGRI下降如果想要停下来放开双手进行其他操作是..
: 在制动端打哑结的。
: 在petzl关于GRIGRI的介绍中，第一句就是“The GRIGRI is a belay device with a..
: ted braking”.说明GRIGRI只是一个能辅助制动的保护器。
: 我们来看一看GRIGRI的制动原理，在攀爬人员坠落时，通过给制动端施力，凸轮受到..
: 的力矩转动继而压住绳子辅助制动。我们看看顺利制动的几个必要条件。
: http://bbs.sjtu.edu.cn../htm/pics/14272574785130.png
: 1.绳子直径在指定的范围内，这个在GRIGRI上就有，如果绳子太细根本不能被压住或..
: 了也不能提供足够的制动力。
: 2.绳子安装方向，安装反了凸轮根本不起作用。
: 3.凸轮的转动是需要制动端施力的，这个容易明白吧？否则绳子可能只是在其中滑过..
: 然有时绳子的摩擦和制动端绳子的自重也可能使凸轮转动而制动，这就是上文说到的..
: ，但这只是可能的情况，仍存在很大的风险。
: 4.凸轮要能够自由地转动，想一想攀爬人员坠落而保护人员一紧张而用力握住了GRIG..
: 绳子的攀爬端，都会卡住凸轮使其不能转动而无法压住绳子制动。因为保护人员紧张..
