: 大家好我就是那个第一个落水的倒霉孩子。
: 拖了蛮久的总结，不过大概因为第一次出线，印象依然很深刻吖，虽然很可能是出于..
: 到了毒奶的威力吧。写得比较碎比较跳跃，而且比较多碎碎念。
: 去的车上玩了小天使与小主人，我们小组大概是画风比较清奇，统一口径希望我们的..
: 使给送一些吃的，虽然后来我们四个人的小天使一个都没有出现（尬）。路上司机师..
: 诉说大明山在下雪，气温大概是零下五度，当时心里很期待雪，但又感觉有点凉，担..
: 己保暖衣物没带够（毒奶从这里开始）。
: 凌晨到的当地向导家，小雨夹雪，铺好防潮垫睡袋就休息了。第一次睡睡袋，感觉把..
: 裹得像毛毛虫一样会睡得很温暖，但是事实证明这完全是我的错觉。当时脑子里是很..
: 的单曲循环着葫芦娃入睡，大概心里其实是觉得自己裹得不像是毛毛虫而像是葫芦娃..
: 凌晨快三点的时候被冻醒，听着不知道谁发出来的呼噜声撑到了3：40左右没忍住掏..
: 发了条票圈“凉凉”，正式毒奶就是从这里开始的，自己之后简直完美印证这两个字。
: 过第一个溪点的时候有点虚，也还不太会用登山杖，有点怕自己一个没踩稳就滑下去..
: 但是过完就觉得，啊没什么很简单不慌不慌，然后第二个溪点就很顺利的落水了……..
: 我本来走很稳的！登山杖当时是卡在石头的一个凹槽处，然后就很放心的撑着要过去..
: 果过到一半的时候应该是因为用力角度变了登山杖就突然滑出来了，然后人就跟着登..
: 下去一屁股坐水里了，但其实没什么事，上来之后也没什么感觉，就是稍微有点点懵..
: 外也幸好领队抢救及时，睡袋基本没湿什么。也是自嘲一下，人在河边走哪能不湿鞋..
: 之后上山过程中过了一些难点，但感觉因为有布绳所以其实没什么问题，主要是因为..
: 了的关系但是有部分路温度还没到太低所以湿哒哒的，一些攀爬点直接导致抓绒手套..
