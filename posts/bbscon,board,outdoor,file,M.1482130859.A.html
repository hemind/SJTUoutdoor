【 在 FANSKY 的大作中提到: 】
: 第一次带没有向导的干事线，先声明本次线路不能作为以后三尖带线的参考，因为这..
: 首先线路准备期间，提前两周与向导联系好，沟通好住宿问题以及最后落脚点位置等..
: 考虑的因素：
: 1三尖需要准备多条下撤线路，不过这个线路本是想准备走完的，希望大家状态也好..
: 2 气温较低
: 3 对干事的要求，本次线路着重提出来是冬训线，对干事有 时间意识，低温下的营..
: 4 体侧，嗯，作为领队在体侧的时候成绩这么差，额，实属不该。。大二的时候记得..
: 5 准备会，会上大家都比较认真，提问考虑也很全面，小插曲是我建议大家带羽绒，..
: 6发装备，虽然当时提示了发厚睡袋，但是貌似大多数人还都是薄的，领队组的装备..
: 走线：
: 人少走起来就是爽啊~~~~~
: 六点庙门集合 ，除了个别大佬迟到时间稍微长一点，欠几百个俯卧撑。。大家都提..
: 到向导家10min之内迅速睡觉，不得不说，这些大部分都没有出过线路的干事们表现..
: 线路过程没有什么太多有趣的事情，岔路并没有很多吧，航迹上直接一看还是很明显..
: 与压队商量翻过绝望坡后找了一个避风的地方吃午饭，随后看来下午的路程断言两个..
: 当晚与压队商量让大家提前起床，但是当晚气温已经很低，领队组做完饭，我就直接..
: 第二天一路上也没什么好说的，就是走到浮桥村后和司机联系，司机晚到10min左右..
