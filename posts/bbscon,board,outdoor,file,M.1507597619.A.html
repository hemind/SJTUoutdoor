: 2017.10.1巨龙顶押队总结
: 开学初登山队长跑来问我，说国庆有三条线，巨龙顶、金紫尖和括苍山，问我要押哪..
: 。因为感觉金紫尖和括苍山对自己都没有什么激励作用，就毫不犹豫选了巨龙顶。这..
: 一的时候走的第一条线，当时啥也不懂，一路跟着闷头走，记忆已经不太清晰，后来..
: 料的时候，感觉要很好的地完成这条线并不容易。事实也证明，尽管做了多次领队押..
: 这条线还是给了我一些新的教训。
: 首先是关于司机疲劳驾驶。这次由于国庆堵车，高速上基本成了停车场，从高速上下..
: 面道路也没有好到哪里去，因此金紫尖和巨龙顶都比预期车程长了很多。而巨龙顶由..
: 途车子坏了修了将近一个小时，加之本身就比较远，到了村子附近还走错了路，所以..
: 更长。而且据我们的师傅说，出发当天下午计师傅还拉着他去跑了另一趟生意。这时..
: 择就非常尴尬了，如果要求司机师傅停在服务区休息，以当时的状况，基本就等于直..
: 消了这条线路；如果像这次实际的一样，稍做休息后接着赶路，那么实际上是非常危..
: ，很有可能出现事故，安全到达只能说是幸运。今后这样的情况在十一也许能够通过..
: 出行日期避免，但是清明、五一的时候还有可能会碰到，这时候领队组该如何抉择，..
: 的部长会或许可以讨论一下，有一个统一的意见。
: 还有就是关于喝水的问题，巨龙顶这次的线路走的时间非常长，过程也很艰难，大家..
: 过量和水量储备不足的问题也暴露无遗。第二天早上很多组都出现了饮用水不足的情..
: 有几组的饮用水平均不足400ml，而第二天行走的时间实际上达到了10.5个小时。饮..
: 备不足的情况还导致了队伍整体士气的低迷，有一些队员甚至表示怀疑能否走下山。..
: ，对于巨龙顶这样第二天仍然十分困难的线路，领队组在第一天晚上做饭的时候一定..
