    颁奖典礼开始咯~ 来点旅行之最，呵呵：）

最佳领队：    
    此奖非solaji莫属咯！自始至终走在队伍最前面领队，探路，休息时间跑前跑后（我
们都气喘吁吁的，却看他谈笑风生），关键时刻果断行事，下山联络。总之，责任心极强
，有老大作风，一路辛苦。

最“牛”领队：
    哈哈，这个奖谁也拼不过这名副其实的牛领队。上山的途中，一头可爱的牛牛居然走
在队伍的最前面，solaji被无视了。不过当后面的驴友们纷纷说着要把登山包交给咱可亲
可近的牛领队背负时，人家的“牛脾气”居然上来了，停在路旁不动了，心想：“哼，不
领队了，我压队。”最终还是人家牛，脱离组织自己玩去了。

最甜的水果：
    下山到农庄，吃到村民的苹果，好脆好甜。虽然大家都催促着时间不多了，没吃饭的
快去吃，我却还想着“能不能再吃一个呢!”

最香的饭菜：
    最后一批上桌吃饭，桌上的五个菜早已。。。呵呵，可就是浇着残余的一点点汤汁，
也能狼吞虎咽的搞定一碗饭，很香，真的很香。旁边的mm说没吃饱，恩，的确如此。

最美味的姜汤：
    从小喝姜汤都是老大难，第一次发现加着果汁的姜汤如此美味，一口下去，一股暖流
通便全身，还想喝，不过不好意思，还是留给别人吧！

最美的星空：
    城里是永远看不到这样的星空的，记得上一次看到这样的星空还是高中在乡村学农的
时候。抬头仰望如此闪亮的星，嘴里不自觉地就哼着“手牵手 一步两步三步四步 望着天
 看星星 一颗两颗三颗四颗 连成线 背着背默默许下心愿 看远方的星是否听的见 它一定
实现”

最可爱歌曲：
    两位国际友人的歌曲大家还记得么？我也不知道怎么翻译了， my English is so po
or，暂且直译吧“two little tigers, two little tigers, run so fast, run so fast
…” 

最深情歌曲：
    应该是xusong的吧，呵呵，据说整首曲目下来一直紧闭双眼，那个忘情啊！可惜没看
到，有人拍pic么？曝一下吧，呵呵：）

最有力量的歌：
    《团结就是力量》，会有人忘记么？夜行路上鼓舞着我们，下山途中振奋着我们。而
我们一路上也一直都秉承着这个原则，服从领队，团结就是力量，坚持就是胜利。

最厚实的手：
    当我们要原路返回找到宿营地时，路滑下山难，而上山更难。那一路上，一双双厚实
有力的手伸向我们，solaji，xusong，徐健，还有许多叫不上名字的朋友们。交大的男生
赞，参加野外生存活动的男生更赞，你们永远值得信赖。

最温暖的手：
    在我左腿膝盖和右脚脚踝受伤后，两条腿完全借不上力，下山一路拄着两只拐杖。有
一段路很窄，走在我后面的应惟伟没有嫌我的脚步慢，几次伸出手支撑着我，很温暖，到
心里的温暖。后来，德国的朋友也赶上来伸出手 in order to help me, 野外生存，不分
国界，不分性别，所有人都是患难与共的朋友。

最佳巾帼：
    xinowen，绝对有着ne女生的豪气，巾帼不让须眉，最后老大下山的一段路上她还一直
作为探路者呢，赞不？

最可爱mm：
    何清秋，呵呵：）这个小mm下山时一直以一个两岁娃娃的角色和她的“Dady”进行口
舌战，一会大喊，一会高歌的，真有精力！

最调皮人士：
    石门林场的小dd。第一批吃过早餐后的我想去拍拍村庄的风景，当我走在小路上突然
感到后面有东西碰了我一下，回头一看，一个4-5岁的小娃娃居然拿着一只竹条抽我pp，随
后他吐个舌头做个鬼脸就跑了。回去张利她们也说，刚刚有个小dd一直想调戏她们，这小
孩子，还真是调皮！

最羞涩人士：
    林场工棚的大叔。吃过中饭休息大家聊天，拍pic，我想和大叔拍一张，征求他意见，
可他说啥都不让，然后就一直躲着我，无奈了：（

最勤奋人士：
    两位国际友人一路向我们学习汉语，大家有目共睹吧，出行还带着德汉字典呢！专心
学习，有照片为证哦！

最可怜人士：
    天黑下山途上，路滑难走，xyhu意欲英雄救美，不幸遗失眼镜。幸好还带了一副有度
数的墨镜备用，不过这也正好满足了他耍酷的小心理，呵呵：）

最佳负重人士：
    辛苦的raowei学弟，有两段路都因为我背负2个登山包，sigh~ 对不起，jj我一定bg你
。

最伤残人士：
    送给我了，大家有异议么？

最灵异事件：
    神雕大侠xusong在内裤中发现的枯叶。呵呵：）

玩、更令人感动的事情发生。这里只是抛砖引玉，希望大家多来补充，呵呵：）
