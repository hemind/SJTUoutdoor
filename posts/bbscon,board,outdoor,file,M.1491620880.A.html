: 一 出发前的准备
: 1首先是这次写的注意事项，发在领队组的群里，由于我在卢湾不能出席准备会，拜..
: 在准备会上叮嘱。
: 1）.有蚂蟥（普及一下处理蚂蟥的知识）
:    （1）会给每个组配酒精
:    （2）队医处准备盐，到营地有蚂蝗要用盐杀死。
:    （3）自己可以自己在鞋面上涂肥皂
:    （4）稍微长点的袜子可以扎住裤脚
: 2）长袖速干衣，有密林
: 3）减好脚趾甲和手指甲
: 4）每组备防晒霜（不要带太大的，最好不要瓶装的）
: 5）有过敏史的，有晕车的请提前报上来
: 6）女生估计一下自己的生理期，可以做适量的准备。
: 2然后是拜托WSY 在医疗包里放一瓶芦苇膏和分装好的四瓶100ml的酒精（每组一瓶）..
: 包盐散。（本来是说20包的，不过后来觉得的确太多了，改成12包）我给每个组分了..
: 左右的盐，并自己备了300克盐，用于营地和在休息地杀死蚂蟥。
: 二 行进用药
