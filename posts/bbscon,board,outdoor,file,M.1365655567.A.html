【 在 Mammut 的大作中提到: 】
: http://www.rock-lizard.org/bbs/thread-13076-1-1.html
: 亮点一：巨石攀岩馆
: 目前上海最好也是最新的抱石馆，只有抱石房，奢华美观；岩点都是国际大赛标准的..
: 司制造，手感极好。缺点就是远了一点儿。。（在浦东，单程地铁就要100分钟。。）
: 亮点二：新赛制（Dark Horse）
: 男女子公开组预赛选手在10:00-14:00之间可以任意攀爬所有预赛线路，不计尝试次..
: 记奖励点，只记所完攀线路的分值。取最高分值的6条线路的积分总和计算成绩排名。
: http://www.rock-lizard.org/bbs/thread-13125-1-1.html
: 线路设定：
: 4条V0      每条分值：1分
: 4条V1      每条分值：2分
: 4条V2      每条分值：3分
: 4条V3      每条分值：4分 
: 4条V4      每条分值：5分
: 4条V5      每条分值：6分
: 4条V6      每条分值：7分
: 简单的说，不用在隔离区等待，想爬哪条跟记分员说一声就可以开始，预赛可以观看..
: 怎么爬~ 而且抱石难度从最简单的到最难的都有，每个童鞋都可以一展身手！！（作..
