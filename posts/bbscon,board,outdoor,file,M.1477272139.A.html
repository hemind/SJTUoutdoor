:   本次作为队医其实还是有点虚的，毕竟以前都是在江浙，山的海拔偏低，第一次去..
: 海拔高强度的山走线，虽然提前做了功课，但还是因为内心很虚，药品都是尽量往多..
: 准备的。还好药品都不重，也就没有出现把队医压挫的情况。反倒是上山前下意识塞..
: 里的一瓶头孢起到了作用。
:   然后，在山上的时候吧，基本上没有累挫掉，已经符合预期了。因为充电宝快没电..
: 也没录航迹。对队员的感觉呢，就是老大哥们很靠谱，小鲜肉嘛好好练很有升值空间..
: 队大人没得说啊。至于领队总结里那个大三童鞋嘛，我得吐槽他一下，我山上用掉的..
: ，一半都用在他身上了（剩下的一半的一半用在我身上了……sigh）。
:   有件事情是一定要自我反省+悔改的，就是对自己身体状况估计不足，带着例假上..
: 事以后再也不做了。一路上靠着一天两顿头孢+伤风停把感冒压下去的，现在想想很..
: ，要是感冒了又高反了，再拖慢了队伍节奏，那真是得不偿失的。
