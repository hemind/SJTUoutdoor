我有一颗敢于做挫事的处女心
【 在 yyhh 的大作中提到: 】
: 以一条浸浴着雨水和鲜血的七尖来作为一段生活的结束真是再合适不过了。
: 先上总结。
: 大约是出发前一周确定当领队的，准备的其实略有仓促，也并没有投入相当的精力做..
: 工作，导致后期还是多多少少出现了疏漏。
: 出发前：
: 1. 报名&队员确定   这一部分的工作相对还是比较顺利且并没有太大的问题。报名..
: 到50人左右，也为后期的队员筛选留下了相当的空间。体能测试也保证了留下的队员..
: 和素质。对于一般的会员线，跑圈这一体测的形式还是比较合理的，一方面易于组织..
: 一方面对身体素质的反映还是相当有说服性且公平公正的。
: 2. 准备会  准备会上的组织多少出现了混乱。首先是收费与身份登记，副领和财务..
: 分工有些凌乱。其次是准备会上的强调点，我其实没有做很细致的准备，除了线路以..
: 本是想到什么讲什么了，难免有所遗漏，比如蚂蝗的事情。
: 3. 装备分发 由于大一考试的缘故，装备分发进行的比较仓促。一个比较重要的遗漏..
: 醒、帮助大家检查各自装备，尤其是帐篷的部件。这一遗漏的后果是很多装备的问题..
: 出行后才发现。出问题的装备大约包括：一个很搓的背包，腰扣后来崩坏；一个帐篷..
: 地钉；一个帐篷的外帐与内帐大小不符合。一方面，没有提醒大家检查是领队组的责..
: 另一方面，装备部也应当相应地反思为什么这些有问题的装备没有做登记且被借出去。
: 4. 出发  一个意外的状况是原先预定的司机一直没有联系上，结果由纪师傅更换司..
: 了一个小时才出发。这一点本身并不是我们的疏漏，就不加以强调了。
