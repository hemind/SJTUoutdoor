【 在 jinqingxu 的大作中提到: 】
: 训练时间：19点到22点20
: 训练地点：南体
: 训练人员：zzm,小蚊子，西瓜，何敏，大爷，庞博，孙瑞田，徐瑾卿
: 训练内容：
: 1、打结
: 复习基本绳结的打法和作用。上周也复习过，一周没练习了部分绳结还是有点生疏，..
: 还是要多加练习。
: 2、热身：3圈慢跑+1次Tabatax
: 3、6圈大变速分为两组 1组3圈
: 因为以前慢跑后变速跑冲不起来，这次的变速跑放在慢跑之前。前3圈还是冲得起来..
: 了后3圈还是非常累，冲不太动。
: 4、4组Tabatax
: 6圈变速对体力消耗特别大，Tabatax每做完一组都瘫倒在地上。
: 5、20圈慢跑 男 2分10 女 2分20
: Tabatax做完以后腿非常酸，特别是弓箭步。慢跑的第一第二圈脚几乎抬不起来，没..
: 正常的姿势跑步。前两圈跑得特别慢，2分50一圈。第三圈开始慢慢调整姿势，尽可..
: 腿，恢复正常的跑步姿势，速度控制在2分30左右。11圈以后又开始感觉力气用尽，..
: 开始自我激励，脑子里想一些好吃的，不断地数还剩下的圈数。到了17圈，我尝试着..
: 步子加速，极限训练的魅力就在这里，当身体累得不行，到处都痛得不行的时候，坚..
: 段时间，后面会感到体力上了一个level,反而更加轻松。
