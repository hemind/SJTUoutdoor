鞋：LOWA Renegade GTX 中帮徒步鞋 1100，八佰伴年末活动拿下。强烈推荐这个鞋
冲锋衣：northland gore tex，两件套， 1100，同上。
包： deuter 28L小包一个，450 淘包。大包考虑买个osprey的60L的苍穹
睡袋： 5度 羽绒睡袋，淘宝买的，400多吧。
其他都是迪卡侬或者杂牌的速干抓绒之类的。


【 在 ianmalcolm 的大作中提到: 】
: 小弟想麻烦各位野人晒一下自己的装备
: 所晒装备属性可以包括但不必限于以下属性：
: 装备名：footwear for hiking
: 品牌型号：Asolo FSN 95 GTX
: 购买地点：淮海路XXX门店
: 成交价格：1500元
: 所晒装备种类可以包括但不必限于以下种类：
: 鞋
: 包
: 衣
: 裤
: 希望这不是一个攀比贴，大家不一定要将自己用过的最贵的装备拿出来晒。又例如有..
: 同学应该不止用过一双鞋吧，你也可以把自己用过哪些鞋或其他装备写出来晒给新人..
: 这些信息对新人认识、了解、购买、使用基本装备具有普遍的重要的参考价值。
: 谢谢各位野人前辈啦
